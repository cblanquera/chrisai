# Brand Research Report

Use this reference when `workflows/brand-research.md` needs a report template,
competitor profile shape, or evidence standard.

## Evidence Standards

Every material finding should be traceable to one of:

- official source URL
- third-party source URL
- user-provided context
- clearly labeled inference from cited evidence

Use these labels:

- `Fact`: directly stated by a source.
- `Quote`: exact source language, kept short.
- `Inference`: agent synthesis from evidence.
- `Unknown`: not found or not checked.

Use confidence levels:

- `High`: multiple current sources or direct official evidence.
- `Medium`: one official source or multiple weaker sources.
- `Low`: limited, stale, unclear, or single-source evidence.
